Using Console
-------------
1) Run the program, when prompted enter the filename in ListOfMazes folder you wish to test. e.g. "large_input.txt"
2) Output will be printed to the console, if you wish to test another file please press any key then re run the program

Using Visual Studio
-------------------
Test -> Run -> All Tests will allow you to run the test scripts which test every file in the ListOfMazes folder.
